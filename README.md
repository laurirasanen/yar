# Yet Another Renderer

A renderer written in modern C++ and Vulkan.

- C++23
- Vulkan 1.4
- SDL3
- Dear ImGui overlay
- Slang shader compile at runtime
- GLTF scenes
- Frustum culling
- Metallic/Roughness PBR
